# AI Fatigue Detection System

This project detects user fatigue using webcam and computer vision.

Features:
- Face and eye detection using OpenCV
- Detects fatigue when eyes remain closed
- Sends email alert to the user
- Burnout prediction using Machine Learning
- Tracks burnout history in Excel
- Web interface using Flask

Technologies Used:
- Python
- Flask
- OpenCV
- Scikit-learn
- HTML
- JavaScript

The system can run as a web application and monitor fatigue through the webcam.
